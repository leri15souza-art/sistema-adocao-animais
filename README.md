# Sistema Web para Adoção de Animais

Aplicação web desenvolvida com React.js para auxiliar ONGs, protetores independentes e pessoas interessadas em adoção responsável de animais.

## Tecnologias utilizadas

- React.js
- HTML JSX
- CSS
- Bootstrap
- Vite

## Funcionalidades

- Exibição dinâmica de animais disponíveis para adoção
- Interface baseada em componentes React
- Cards responsivos
- Formulário de contato
- Layout responsivo

## Como executar o projeto

1. Instalar dependências:

npm install

2. Executar o projeto:

npm run dev
